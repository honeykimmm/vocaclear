# 보카클리어 고교필수편 — 사용 안내

## 지금 바로 쓸 수 있는 것
파일을 열면 바로 작동해요. 단어 목록 / 플래시카드 / 타이핑테스트 / 내 점수 기록(이 기기 한정)까지 전부 됩니다.
다만 Firebase 설정을 하지 않으면 "반별 실시간 랭킹"과 "교사 대시보드"가 **이 기기에만 저장된 기록**만 보여줘요.
(다른 학생 휴대폰에서 푼 결과가 안 보임)

## 반 전체가 공유하는 실시간 랭킹을 켜려면 (10분이면 됩니다)

1. https://console.firebase.google.com 접속 → 구글 계정으로 로그인 (학원 계정도 OK)
2. "프로젝트 추가" → 이름 아무거나 (예: vocaclear) → 애널리틱스는 꺼도 무방
3. 왼쪽 메뉴에서 **Firestore Database** → "데이터베이스 만들기" → **테스트 모드로 시작** (나중에 보안 규칙 강화 가능)
4. 왼쪽 위 톱니바퀴 → **프로젝트 설정** → 아래로 스크롤 → "내 앱" → `</>` (웹 앱 추가) 클릭
5. 앱 닉네임 아무거나 입력 → 등록하면 `firebaseConfig` 라는 코드 블록이 나와요. 이 안의 값들을 복사

6. `firebase-config.js` 파일을 열고, 맨 위 `firebaseConfig` 객체를 방금 복사한 값으로 통째로 교체

```js
const firebaseConfig = {
  apiKey: "여기에 본인 값",
  authDomain: "여기에 본인 값",
  projectId: "여기에 본인 값",
  storageBucket: "여기에 본인 값",
  messagingSenderId: "여기에 본인 값",
  appId: "여기에 본인 값"
};
```

7. 저장하고 새로고침하면 끝! 이제 학생들이 각자 폰에서 풀어도 같은 반이면 실시간으로 순위가 갱신돼요.

## 반 이름 / 비밀번호 바꾸고 싶으면
`app.js` 맨 위쪽에서 수정하면 돼요.

```js
// 선생님 > 반 구조
const TEACHER_CLASSES = {
  "강민지": ["월수반", "월금반", "화목반"],
  "송정은": ["화목반"],
};

// 교사 대시보드 비밀번호 (전체 공유)
const TEACHER_DASH_PASSWORD = "dyb2024";
```

선생님을 추가하거나 반을 늘리고 싶으면 위 `TEACHER_CLASSES` 객체에 줄을 추가하면 자동으로 학생용 선택 화면과 랭킹/대시보드에 반영돼요.

## 앱처럼 설치해서 쓰기 (PWA)

이 앱은 PWA(Progressive Web App)로 만들어져서, 휴대폰 홈 화면에 추가하면 진짜 앱처럼 아이콘을 눌러 바로 실행돼요.

**처음 1회**는 인터넷이 연결된 상태에서 접속해야 해요(파일들을 기기에 캐싱하는 과정). 그 이후로는 단어 목록, 플래시카드, 타이핑테스트 전부 **인터넷 없이도** 작동해요. (타이핑테스트 결과는 오프라인 중엔 기기에 임시 저장됐다가, 다시 인터넷이 연결되면 자동으로 서버에 동기화돼요.)

### 설치 방법
- **아이폰(Safari)**: 사이트 접속 → 하단 공유 버튼(⬆️) → "홈 화면에 추가"
- **안드로이드(Chrome)**: 사이트 접속 → 우측 상단 점 세 개(⋮) → "홈 화면에 추가" 또는 자동으로 뜨는 설치 배너 사용

### 발음 듣기 기능 안내
발음은 휴대폰에 내장된 음성 엔진을 사용해요. 안드로이드는 대부분 오프라인에서도 잘 동작하고, 아이폰도 기본으로 깔린 영어 음성이면 오프라인에서 됩니다. 다만 기기/OS 버전에 따라 일부 음성은 첫 사용 시 다운로드가 필요할 수 있어요 — 한 번이라도 인터넷 연결 상태에서 발음을 들어본 단어는 그 다음부턴 오프라인에서도 잘 됩니다.

### 내 아이콘으로 바꾸기
직접 만든 아이콘으로 바꾸고 싶으면, 아래 두 파일을 같은 이름으로 덮어쓰면 돼요.
- `icons/icon-192.png` — 192×192px
- `icons/icon-512.png` — 512×512px

정사각형, 배경은 투명 대신 단색으로 꽉 채워주세요(투명 배경은 기기마다 처리 방식이 달라요). 그림은 가장자리에서 살짝 안쪽에 배치하면 둥근 모서리로 잘려도 안전해요.

### 앱 업데이트할 때 주의
나중에 코드를 수정해서 다시 올리면, 이미 설치한 학생들 폰에는 자동으로 바로 반영되지 않을 수 있어요(서비스 워커 캐시 때문). 그럴 땐 `service-worker.js` 파일 맨 위 `CACHE_NAME` 값을 `"vocaclear-v2"`처럼 바꿔서 같이 올리면, 학생들이 앱을 다시 열 때 새 버전으로 자동 교체돼요.


1. 타이핑테스트 탭 → 이름 입력 → 선생님 선택 → (선생님 반이 2개 이상이면) 반 선택
   - 송정은 선생님은 화목반 하나뿐이라 선생님만 선택하면 반이 자동으로 선택돼요
   - 한 번 입력하면 이 기기에 저장돼서 다음부턴 자동으로 불러와요
2. 풀고 싶은 DAY 선택 (여러 개 선택 가능) → 시작
3. 뜻 보고 영단어 입력, Enter로 다음 문제
4. 결과 화면에서 "내 기록 보기" 누르면 그래프와 추세 멘트 확인 가능 → 이미지로 저장도 가능

## 교사 사용법
"교사 대시보드" 탭은 비밀번호를 입력해야 들어갈 수 있어요 (학생들이 못 보게 보호됨).
들어가면 선생님/반별로 학생들의 응시 횟수, 전체평균, 최근 3회 평균, 상승/하락 추세를 한 번에 확인할 수 있어요.
다 보고 나면 우측 상단 "🔒 잠그기"를 누르면 다시 비밀번호 화면으로 돌아가요. (브라우저를 완전히 닫았다 열어도 자동으로 잠겨요)

## 디자인
다크모드/라이트모드는 휴대폰 시스템 설정을 자동으로 따라가요. 헤더의 ◐ 버튼을 눌러서 자동/라이트/다크를 수동으로 바꿀 수도 있어요.
